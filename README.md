# 🌾 Agro Forte, Futuro Sustentável - Nova Tebas

Uma página web institucional e acessível dedicada a apresentar a força, a tradição e a sustentabilidade do agronegócio e da agricultura familiar no município de Nova Tebas, Paraná.

---

## 🚀 Funcionalidades Principais

*   **Design Responsivo e Fluído**: Desenvolvido com técnicas modernas de CSS (`clamp()`), garantindo que textos, títulos e espaçamentos se adaptem perfeitamente a qualquer tamanho de tela (monitores grandes, tablets e celulares).
*   **Painel Flutuante de Acessibilidade ♿**:
    *   **Controle de Fonte**: Botões para aumentar (+) ou diminuir (-) o tamanho de todos os textos da página de forma proporcional.
    *   **Modo Escuro (Dark Mode)**: Alternador de tema para melhorar o conforto visual em ambientes de baixa luminosidade.
*   **Abas Interativas (Accordion)**: Seção inteligente "Pilares da Nossa Produção", que organiza as cadeias produtivas (leite, grãos, corte, orgânicos e agroindústrias) em blocos expansíveis com animações suaves controladas por JavaScript.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído do zero utilizando tecnologias web nativas (sem a necessidade de instalar frameworks ou bibliotecas externas):

*   **HTML5**: Estruturação semântica do conteúdo e marcações de acessibilidade (`aria-labels`, `aria-expanded`, `aria-hidden`).
*   **CSS3**: Estilização moderna com variáveis globais (`:root`), transições suaves e tipografia dinâmica.
*   **JavaScript (ES6+)**: Lógica para manipulação do DOM, controle dos estados do accordion e interações do painel de acessibilidade.

---

## 📂 Estrutura de Arquivos

Para garantir o máximo desempenho e facilidade de carregamento, o projeto foi unificado em um único arquivo estrutural:

```text
├── index.html       # Arquivo principal contendo a estrutura HTML, estilos (CSS) e lógica (JS)
```

---

## 💻 Como Executar o Projeto

Como o site utiliza tecnologias nativas do navegador, você não precisa instalar nenhuma ferramenta de servidor.

1. Baixe ou copie o código do arquivo `index.html`.
2. Dê um **duplo clique** no arquivo `index.html` ou arraste-o para dentro de qualquer navegador web (Google Chrome, Firefox, Edge, Safari).
3. Pronto! A página estará rodando localmente no seu computador.

## ♿ Recursos de Acessibilidade Implementados

Este site foi planejado com foco na inclusão digital, seguindo boas práticas de desenvolvimento:
*   Uso de tags semânticas (`<header>`, `<main>`, `<section>`).
*   Atributos ARIA atualizados dinamicamente via JavaScript para leitores de tela quando as abas abrem ou fecham.
*   Fechamento automático do menu flutuante ao clicar fora dele para evitar poluição visual.
